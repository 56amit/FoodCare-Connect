✅ **Full Project Summary (in paragraph):
FoodCare-Connect is a socially impactful full-stack web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). It connects food donors (like individuals, restaurants, and wedding halls) with NGOs who distribute the food to needy people. The platform includes three main roles: Donor, NGO, and Admin. Donors can submit food details with images, NGOs can accept and manage donations, and the Admin can oversee the entire system. The application includes secure authentication using JWT and Bcrypt.js, image upload using Multer with Cloudinary integration, real-time email notifications, and a fully responsive UI. It aims to reduce food wastage and support hunger relief through a streamlined and modern solution.

🔧 Frontend:
React.js (with Vite) – for fast and optimized UI development.

TailwindCSS – to create a fully responsive and clean design.

React Router DOM – for client-side routing/navigation.

Axios – to handle communication with backend APIs.

Context API – for managing global state like user login/auth state.

🛠️ Backend:
Node.js with Express.js – to handle server-side logic and routing.

MongoDB with Mongoose – as the database and ORM for managing food, user, and donation data.

Multer – to handle image uploads from users.

Cloudinary – for storing uploaded food images securely in the cloud.

JWT (JSON Web Tokens) – for secure user authentication.

Bcrypt.js – to encrypt and hash user passwords.

✨ Key Features:
Role-based login system: Donor, NGO, Admin.

Food donation with image upload and location.

Email notifications on food request and approval.

Admin dashboard to manage users, NGOs, and donations.

Fully mobile-responsive interface.

Real-time status updates for donation tracking.
